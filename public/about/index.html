    新墨韬

    关于本站#

    新墨韬是一个个人技术博客，名字取"笔墨韬略"之意 —— 用文字记录技术路上的思考与实践。

    这里主要分享以下内容：

    • Hugo 静态网站：主题开发、配置优化、部署方案
    • 开源工具：Git、Docker、Linux 等开发工具的使用经验
    • 编程实践：Python、Go 等语言的实用技巧
    • 工作流方法：现代软件开发的高效实践

    博客基于 Hugo 构建，使用自研的 PaperUI 主题。PaperUI 在 PaperMod 的基础上进行了深度定制，增加了首页轮播、双栏布局、彩色标签系统、搜索增强等特性，专为中文技术博客优化。

    技术栈#

    组件 技术选型 说明
    静态生成 Hugo Go 语言编写，构建速度极快
    主题 PaperUI 基于 PaperMod 深度定制的优化版
    托管部署 Cloudflare Pages / GitHub Pages 全球 CDN 加速，免费 SSL
    图片资源 WebP + JPEG 按需使用，优先 WebP 格式

    为什么写博客#

    • 整理知识：写作是最好的学习方式，把零散的知识结构化
    • 分享经验：踩过的坑、总结的技巧，或许能帮到遇到同样问题的人
    • 保持思考：定期输出倒逼持续输入，形成正向循环
    • 开源精神：代码开源，文章也开源 —— 本站所有内容在 GitHub 上可见

    关于 PaperUI 主题#

    PaperUI 是这个博客使用的主题，也是我开发的开源项目。如果你也在用 Hugo 搭建博客，可以试试 PaperUI：

    • 保留 PaperMod 的全部核心能力（SEO、性能、多语言）
    • 新增首页轮播、侧边栏、彩色标签、搜索增强等实用功能
    • 对中文博客场景做了深度本地化优化
    • MIT 协议开源，自由使用和修改

    好的工具让创作更专注，好的主题让内容更闪耀。
