    Hugo 快速入门指南

    June 1, 2026 · 1 min · 31 words

    Hugo 是一个用 Go 语言编写的静态网站生成器，以其构建速度快而闻名。

    安装 Hugo#

    Hugo 的安装非常简单，你可以在官方网站下载对应平台的二进制文件。

    基本用法#

    # 创建新站点
hugo new site my-site

# 创建新文章
hugo new posts/my-post.md

# 启动开发服务器
hugo server -D
